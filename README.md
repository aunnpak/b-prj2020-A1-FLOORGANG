How to play:
W = move up
A = move left
S = move down
D = move right
Space = Shooting
R = restart level

Game Mode
1. Endless - Survive as much enemy wave as you can
2. Score   - Collect all lost item in the map
3. Goal    - Travel through any obstacle to the goal
